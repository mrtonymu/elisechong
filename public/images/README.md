3. **about-life.jpg** - About 区域的生活照片
   - 半身照，浅米色夹克

## 📁 备用图片

备用照片存放在 `backup/` 目录中，可根据需要替换使用。

## 图片要求

- 格式：JPG 或 PNG
- 质量：高清，清晰
- 风格：专业、温暖、自然
- 文件大小：建议每张不超过 500KB（可以通过压缩工具优化）

## 使用方式

在组件中，照片路径应该指向 `/images/hero-photo.jpg` 等。
